* Patches

Moving files between repos, preserving their history.

** Create

#+begin_src shell

git log --pretty=email --patch-with-stat --reverse --full-index --binary -- path/to/file_or_folder > patch

#+end_src

** Apply

#+begin_src shell

git am --committer-date-is-author-date < ../repository/patch 

#+end_src
